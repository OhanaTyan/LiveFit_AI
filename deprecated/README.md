# Deprecated Files Directory

此目录包含项目早期开发阶段的参考实现和测试代码，这些文件目前已不再是主项目的一部分，但可能对未来的开发有参考价值。

## 目录结构

```
deprecated/
├── assets/
│   └── aerobic_data.json       # 有氧运动卡路里消耗数据
├── dart_models/
│   └── aerobic_activity.dart   # 有氧运动数据模型
├── dart_services/
│   └── aerobic_data_service.dart # 有氧运动数据服务
└── python_logic_ref/
    ├── scheduler_engine.py      # AI调度器核心逻辑（Python参考实现）
    └── user.py                  # 用户模型（Python参考实现）
```

## 文件说明

### 有氧运动数据相关文件
- **aerobic_data.json**：包含了各种有氧运动的卡路里消耗数据，按运动类型、强度和持续时间分类
- **aerobic_activity.dart**：定义了有氧运动的数据模型
- **aerobic_data_service.dart**：实现了有氧运动数据的加载和卡路里计算功能

### AI调度器参考实现
- **scheduler_engine.py**：包含了AI调度器的核心逻辑，用于根据用户体质和日程冲突生成替代训练计划
- **user.py**：定义了用户模型和认证相关的数据结构
