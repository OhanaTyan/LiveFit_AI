# 项目文档说明

此目录包含LifeFit AI项目的核心文档，描述项目的基础架构、技术栈和设计原则。

## 核心文档列表

| 文档名称 | 主要内容 |
|---------|---------|
| `architecture_design.md` | 详细的架构设计和模块划分 |
| `project_structure.md` | 项目结构说明，包括代码目录和核心文件 |
| `roadmap.md` | 项目发展路线图和优化计划 |
| `technical_architecture.md` | 技术架构和技术栈说明 |

## 其他文档目录

- **docs/development/** - 开发相关的指南和规范
- **docs/api/** - API设计和使用说明
- **docs/features/** - 功能模块文档
- **docs/operations/** - 运维和部署相关信息
- **文档目录说明.md** - 项目文档结构概览（根目录）
- **文件目录结构说明.md** - 项目代码目录结构概览（根目录）

## 文档编写规范

1. 使用Markdown格式编写
2. 中文文档使用中文标点符号
3. 英文文档使用英文标点符号
4. 图片统一放在项目根目录下的合适位置
5. 文件名使用小写字母，用连字符连接多个单词

## 文档维护

1. 新增文档需要在对应README中更新文档列表
2. 修改重要文档需记录变更历史
3. 删除文档需确认不再使用并通过团队review
4. 保持文档与代码同步更新
