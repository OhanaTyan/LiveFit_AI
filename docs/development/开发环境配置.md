# 开发环境配置指南

## 系统要求

- 操作系统: Windows 10+/macOS 10.15+/Linux Ubuntu 20.04+
- 内存: 8GB以上(推荐16GB)
- 硬盘空间: 10GB以上可用空间

## 必要软件安装

1. **Flutter SDK**

```bash
# 下载最新稳定版
git clone https://github.com/flutter/flutter.git -b stable

# 添加环境变量
export PATH="$PATH:`pwd`/flutter/bin"
```

1. **Android开发环境**
   - Android Studio
   - Android SDK
   - 配置环境变量ANDROID_HOME

2. **iOS开发环境**
   - Xcode (仅macOS)
   - CocoaPods

3. **编辑器/IDE**
   - VS Code (推荐)
   - Android Studio
   - IntelliJ IDEA

## 项目依赖安装

```bash
# 安装项目依赖
flutter pub get

# 运行项目
flutter run
```

## 常见问题解决

1. **Flutter doctor问题检查**

 ```bash
 flutter doctor
 ```

1. **Android模拟器配置问题**
   - 确保已启用虚拟化
   - 检查BIOS设置

2. **iOS模拟器问题**
   - 确保Xcode命令行工具已安装

   ```bash
   xcode-select --install
   ```
