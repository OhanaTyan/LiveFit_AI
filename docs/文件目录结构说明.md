# LifeFit AI 文件目录结构说明

## 项目概述

LifeFit AI 是一款日程驱动的智能健身助手，采用 Flutter 框架开发，支持多平台运行。项目采用分层架构设计，具有良好的可扩展性和维护性。

## 核心目录结构概览

```
life_fit/
├── android/            # Android平台相关代码
├── ios/                # iOS平台相关代码
├── lib/                # Dart应用程序代码
│   ├── main.dart       # 应用入口文件
│   └── src/
│       ├── core/       # 核心共享代码
│       │   ├── localization/  # 国际化支持
│       │   ├── services/      # 共享服务
│       │   ├── theme/         # 主题配置
│       │   └── utils/         # 工具类
│       └── features/   # 功能模块
│           ├── authentication/    # 用户认证
│           ├── dashboard/         # 仪表盘
│           ├── habits/            # 习惯养成
│           ├── onboarding/        # 引导流程
│           ├── profile/           # 用户资料
│           ├── schedule/          # 日程管理
│           ├── settings/          # 设置
│           ├── statistics/        # 统计分析
│           ├── voice_schedule/    # 语音日程
│           └── weather/           # 天气集成
├── test/               # 测试代码
├── windows/            # Windows平台相关代码
├── linux/              # Linux平台相关代码
├── macos/              # macOS平台相关代码
├── docs/               # 项目文档
└── pubspec.yaml        # 项目配置文件
```

## 目录结构说明

### 1. 核心目录

- **android/ios/windows/linux/macos/** - 各平台相关代码
- **lib/** - 主要的Dart应用程序代码
- **test/** - 单元测试和widget测试代码
- **docs/** - 项目文档

### 2. 代码组织原则

项目采用清晰的分层架构设计：

1. **分层架构**：清晰的分层设计，分离关注点
2. **模块化设计**：按功能模块组织代码，提高可维护性
3. **依赖注入**：使用 Provider 进行状态管理和依赖注入
4. **响应式设计**：采用 Flutter 响应式框架，适配不同屏幕尺寸
5. **国际化支持**：完整的国际化支持，便于扩展到不同地区
6. **主题定制**：支持明暗主题切换，提供良好的用户体验

3. **代码风格**：遵循 Flutter 官方代码风格指南
4. **注释规范**：关键代码添加注释，说明功能和实现逻辑
5. **测试覆盖**：核心功能编写单元测试和集成测试
